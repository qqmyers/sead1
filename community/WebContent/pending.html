Pending Publication Requests from Project Spaces

This page shows all active publication requests from SEAD's project spaces. It is intended primarily as an aid for curators who can quickly see all pending requests in one place. Note: Being able to see the request does not mean that you will be able to view the collection metadata or view the datasets within it (prior to publication, these are access controlled based on the configuration of an individual space).

You can try this out by going to the Demo Space and clicking the "Submit for Publication" button on a collection(s). Let us know if you would like to have SEAD staff process your request (generate a test DOI, package your data and metada, and submit it to a repository). Note: We may remove content from the demo space, as well as any published data from it, at any time (it is just for testing).
